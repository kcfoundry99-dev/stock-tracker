# Blue Note 唱片公司的故事

1940 年代，紐約哈林區。兩個年輕人，Alfred Lion 和 Francis Wolff，在一間小地下室創立了 Blue Note Records。

Francis 是德國來的猶太人，Alfred 是印刷工人。他們沒什麼錢，但有一個瘋狂的夢想：錄製最真實的爵士樂。

Blue Note 的聲音很特別。低音提琴不是輕輕的伴奏，而是跳動的心臟。鼓不是簡單的打拍子，而是靈魂的脈搏。

他們錄過一個年輕人，叫 Thelonious Monk。這個人的鋼琴太奇怪了，別的唱片公司說他「太前衛」、「聽不懂」。但 Blue Note 說：「不，這就是未來的聲音。」

後來又有一個貝斯手，叫 Charles Mingus。他的脾氣暴躁得像一團火，但他的音樂讓整個爵士樂壇翻轉。

Blue Note 不只是唱片公司。它是一個宣言：真正的藝術不應該被商業綁架。

那個藍色的封面，那個簡單的 logo... 成為了爵士樂的經典標誌。

直到今天，當人們看到那個藍色的封面，還是會說：「啊，這是真正的爵士樂。」
